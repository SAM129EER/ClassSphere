# ClassSphere 🎓

ClassSphere is a modern full-stack classroom management platform built with React, TypeScript, Express, PostgreSQL, and Drizzle ORM.

The platform supports role-based access for admins, teachers, and students with authentication, classroom management, enrollments, and cloud-based media uploads.

---

# 🚀 Features

- 🔐 Authentication & Authorization
- 👨‍🏫 Teacher Dashboard
- 🎓 Student Enrollment System
- 🏫 Classroom Management
- 📚 Subject & Department Management
- ☁️ Cloudinary Media Uploads
- 🛡️ Arcjet Security & Rate Limiting
- 🌙 Dark/Light Theme Support
- 📱 Responsive UI
- 🗄️ PostgreSQL Database
- ⚡ Modern Full-Stack Architecture
